# Mindshot data

This is a personal Mindshot data repository. It is written to by the Mindshot
capture client and read by the Mindshot brain.

- `raw/` — voice captures, one Markdown file per day (`raw/YYYY-MM-DD.md`).
- `derived/` — produced by the brain from the raw captures.

Conforms to **CONTRACT.md v1.0**. The capture client only ever writes to `raw/`.
